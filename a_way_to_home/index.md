# 饭组端午环湖自驾计划

本文记录一下饭组的端午节下乡出游计划（也就是计划一下如何把大伙儿拐进山里面）

### 出发前准备
1. 72 小时核酸检测做一下，确保旅游时间内，核算都在有效期内。
2. 自驾汽车状态检查一下，确保路途自驾安全，自驾来回预计消耗半箱油。
3. 照相机📷，打卡必备。
4. 手机充电线以及充电宝。
5. 一套换洗衣物避免意外需要换洗的场景。
6. 防蚊虫喷雾、防晒喷雾、酒精棉片，碘伏等简易应急物品
